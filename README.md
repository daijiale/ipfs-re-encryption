# ipfs-re-encryption

Some Re-encrytion Demo For IPFS Data


## Intro

- src
  - ipfs.php  (ipfs php api library class)
  - reEncrytion.php (re-encrytion api library class)
- testBase64.php （test script）

## Usage

Install php-5.6

```php
php testBase64.php

```
